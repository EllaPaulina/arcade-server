Adding games

You can add Excalibur, Phaser, Makecode, etc. games by publishing your game live, for example on github pages.

You can add Unity, GameMaker or GoDot games if you export to WebGL.

Steps

  • Publish your html game page online
  • Add gamepad support to your game
  • Make sure no mouse input is needed anywhere (no start button)
  • Ask teachers to add the game OR add it by making a Pull Request!
  • Scan the QR code for more info
Press 🟡 🟡 to go back to the games.
